Privacy Policy

Last updated: March 2026  ·  Musico Technologies

1. Information We Collect

Musico operates by collecting only the minimum Discord-provided data required to function. This includes:

  • Server Data: Server IDs, Channel IDs, and Role IDs for configuration such as auto-mod setup, logging channels, and saved roles.
  • User Data: User IDs to track warnings, infractions, and command usage statistics. Message content is temporarily scanned for auto-mod but is never stored.
  • Dashboard OAuth: When you log in via Discord, we receive your Discord username, avatar, and the list of servers you manage. This data is stored only in your session cookie.

2. How We Use Information

We use the gathered information solely to provide the core functionality of the bot and dashboard. For example: logging warnings to your configuration, executing AI requests via Gemini, running server metrics, and displaying your server list in the dashboard.

3. Data Storage and Security

All structured configuration data (warnings, automod settings) is stored locally on secure JSON files in our hosting environment. Dashboard sessions are managed via secure HTTP-only server-side session cookies. We do not sell or share any user data with third-party companies.

4. Your Rights to Deletion

If you kick the bot from your server, you may use Discord's integration settings. You may also contact our support team to request a full manual wipe of your server's JSON configurations from our database. Dashboard session data is automatically cleared on logout.

5. Changes to This Policy

We may update this Privacy Policy from time to time. We will notify users of any material changes by posting an announcement in our support server or sending a broadcast message.

  If you have any questions about this policy, please reach out via our Support Server.
